# Academic Advising & Progress Tracking System

## Project Overview
This is a Java Web Application built using JSP, Servlet, and MySQL.
It allows students/advisors to manage advising sessions using CRUD operations.

## Features
- Registration & Login
- Dashboard (KPIs and latest sessions)
- Advising Sessions (CRUD: Create, Read, Update, Delete)
- Logout feature

## Technologies Used
- Java (Servlets + JSP)
- MySQL (phpMyAdmin)
- Apache Tomcat
- Bootstrap (UI)

## How to Run the Project
1. Open the project in NetBeans
2. Start Apache Tomcat
3. Import the SQL file into phpMyAdmin
4. Run the project and login

## Database Setup
- Database name: academic_advising
- Import file: academic_advising.sql
