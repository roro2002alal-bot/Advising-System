Registration Form (Placeholder)

This section represents the registration form the real system will use to collect basic information about new users.

  • Full name field
  • University email address
  • Student ID (or Employee ID for advisors)
  • Major / Department selection
  • Role selection (Student / Advisor)
  • Password and Confirm Password fields
  • Register button

Account & Advising Context

In the actual application, a newly registered student would be linked to an academic advisor and their degree plan. The registration data here would be stored in tables such as users and students in the database, and later used to display progress and advising information on the dashboard.

Storyboard Note

This is a static storyboard page only. No real account creation or validation happens here. The goal is to show where registration fields, buttons, and messages will appear in the final Academic Advising & Progress Tracking web system.
