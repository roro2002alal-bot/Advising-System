This storyboard section represents the form for creating a new advising session. No real submission happens here.

  • Student selection dropdown
  • Advisor selection dropdown

  • Session type (Academic Plan, Follow-up, Warning, Graduation Review)
  • Risk level selection (Low, Medium, High)
  • Notes / Comments text area

This form will send data to the advising_sessions table in the real system.

Storyboard Note

In the final application, this page would validate input fields, save the session to the database, and redirect the advisor back to the session list. For the storyboard, it only shows the layout and flow.
