Edit Advising Session Form (Placeholder)

This storyboard section shows the structure of the form used to edit an existing session. The fields would be pre-filled with the current data.

  • Read-only Session ID
  • Student selection dropdown (pre-selected)
  • Advisor selection dropdown (pre-selected)
  • Date and time picker (existing value)
  • Session type selection (existing value)
  • Risk level selection (existing value)
  • Notes / Comments text area with current notes
  • Save Changes button
  • Cancel button (returns to details or list)

In the real system, submitting this form would update the corresponding record in the advising_sessions table.

Storyboard Note

No real update occurs on this storyboard page. Its purpose is to show where editable fields and action buttons will appear, and to demonstrate the flow from List → Details → Edit → List within the advising sessions module.
